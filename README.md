# 📌 Features

* ✍️ Rich text blog editor using React Quill
* 🤖 AI-assisted writing support
* 🌙 Light & Dark mode
* 📱 Fully responsive modern UI
* 📝 Draft saving system
* 🔐 JWT authentication
* 👤 User-specific posts and drafts
* 💬 Comment system
* 🔎 Search functionality
* 📖 Trending & history pages
* ⚡ Framer Motion animations
* 🎨 Tailwind CSS modern design
* ☁️ Full deployment on Vercel + Render

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/kishorrjeyachandran/ThoughtNest.git
```

---

## 2. Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3. Install Backend Dependencies

```bash
cd backend
npm install
```

---

# 🔑 Environment Variables

Create `.env` inside backend folder.

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_openai_api_key
PORT=5000
```

---

# ▶️ Run Locally

## Frontend

```bash
cd frontend
npm run dev
```

## Backend

```bash
cd backend
npm run dev
```
